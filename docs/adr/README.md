# Architecture Decision Records

This directory contains Acumius Architecture Decision Records (ADRs).

## Purpose

ADRs capture what we decided, why we decided it, and what alternatives were considered. They are append-only records. If we change direction later, we create a new ADR that supersedes the old one.

## Naming Convention

- `ADR-0001-core-engine-language.md`
- `ADR-0002-memory-schema-format.md`

## Status Values

- `Proposed`
- `Accepted`
- `Superseded`
